Flashcards

JavaScript

Object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Symbols used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

Variable naming conventions

Style guidelines for programming. They typically cover naming and declaration rules for variables and functions.

Functions

Block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

Methods

Actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Arguments

Values received by the function when it is invoked. Inside the function, the arguments (the parameters) behave as local variables.

Parameters

Names listed in the function definition. Function arguments are the real values passed to (and received by) the function.

Return

Stops the execution of a function and returns a value from that function.

Chaining methods

Allows us to run multiple jQuery methods (on the same element) within a single statement.

Strings

JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

One of two values: true or false.

Undefined

Indicates that a variable has not been assigned a value, or not declared at all.

NaN

Property that represents not a Number

Escape

Tells JavaScript that the character (ex."") that comes right after it is not the end of the string, but just a character inside the string.

Alert

Method that displays an alert box with a specified message and an OK button.

Comments

Comments that can be used to explain JavaScript code, and to make it more readable or to prevent execution, when testing alternative code.

jQuery

JavaScript library.

Attributes

How the property can be accessed (is it readable?, is it writable?).

Property

Values associated with a JavaScript object. Can usually be changed, added, and deleted, but some are read only.
